# Contributing to ebr-mod-registry

This repository is the central directory of community mods for Earthborne
Rangers. Contributions here are **mod submissions** - adding or updating your
mod's entry in the registry so players can find it.

## Submitting a Mod

The correct way to submit a mod is through the `ebr publish` CLI command
from [ebr-mod-tools](https://github.com/Earthborne-Rangers-Community-Mods/ebr-mod-tools). It handles
the entire process automatically:

1. Validates your `ebr-mod.json` manifest
2. Checks that your mod ID is not already claimed by another author
3. Creates or updates `mods/<your-mod-id>.json` on your fork of this repo
4. Opens a pull request against this repository

## Review Process

Every submission requires maintainer approval before it appears in the mod
manager app. Reviewers check:

- Required fields are present and correctly formatted
- The `commitHash` matches a real commit on the `repoUrl`
- The mod ID is not already claimed by a different author
- Content complies with the [Code of Conduct](CODE_OF_CONDUCT.md)

## Updating Your Mod

Run `ebr publish` again after making changes to your mod. It updates your
existing `mods/<mod-id>.json` file with the new version, commit hash, and
timestamp, then opens a new PR.

## Reporting a Mod

If you find a mod that violates community standards, open an issue on this
repository. Include the mod ID, the reason for the report, and any supporting
evidence.

## What Not to Do

- **Do not edit `registry.json` by hand.** It is a build artifact generated by
  CI from the files in `mods/`.
- **Do not modify another author's mod file.** Each mod file is owned by its
  author. Only maintainers remove entries.
- **Do not submit mods with misleading descriptions.** The description must
  accurately reflect the mod's content.

## Questions

Open a GitHub Discussion on this repository for questions about the submission
process, registry format, or moderation decisions.
